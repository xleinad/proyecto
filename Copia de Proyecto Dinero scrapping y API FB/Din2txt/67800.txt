Microsoft quiere cambiar el teclado del iPhone
Ciudad de México.- Dado el escaso éxito obtenido con los Windows Phone, ahora Microsoft decidió cambiar de estrategia, y poco a poco abrir su software para que sea compatible con otros sistemas operativos.
Microsoft llevará el teclado de sus teléfonos al iOS de Apple. De acuerdo con The Verge, la empresa de Bill Gates se encuentra en una fase de pruebas interna del teclado de Windows Phone en iOS, para luego lanzar una versión enfocada a llegar a un grupo de usuarios seleccionado.
Una de las características más atractivas de este teclado, es que tiene una modalidad que permite escribir con una sola mano sin importar el lado con que se sostenga el teléfono.
Se espera que el teclado de Windows Phone llegue a iOS en los próximos meses, mientras que en Android estará listo a finales de año.
